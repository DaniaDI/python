
📌 Singly Linked List Implementation (Python)
🧠 Overview:-

    -This project is a simple implementation of a Singly Linked List using Python.

    -A linked list is a linear data structure where each element (node) contains:
    - A value
    -A reference (pointer) to the next 
    
⚙️ Features:-

This implementation supports the following operations:

Add a node to the front of the list
Add a node to the back of the list
Remove a node from the front
Remove a node from the back
Print all values in the list

🧩 Classes Structure:-
    🔹 SNode

    Represents a single node in the linked list.

    Attributes:

    val → stores the data
    next → points to the next node
    🔹 SList

    Manages the linked list and all operations.

🚀 Methods
    1- add_to_front(val):Adds a new node at the beginning of the list.
    2- add_to_back(val):Adds a new node at the end of the list.
    3-print_values():Traverses the list and prints all node values.
    4-remove_from_front():Removes the first node and returns its value.
    5-remove_from_back():Removes the last node and returns its value.
