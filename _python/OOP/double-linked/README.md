📌 Doubly Linked List Implementation (Python)

🧠 Overview:-

    This project is a simple implementation of a Doubly Linked List in Python.
    Each node contains a value and two pointers:

    next → points to the next node
    prev → points to the previous node

    This allows traversal in both directions.

    Featuers:
    
    Add node to the front of the list
    Add node to the back of the list
    Print all values in the list
    🧩 Classes Structure

🔹 SNode
Represents a single node in the linked list.

Attributes:

    val → value stored in the node
    next → pointer to the next node
    prev → pointer to the previous 

🔹 SList
Represents the linked list and manages all operations.
 

🚀 Methods
1- add_to_front(val):Adds a new node at the beginning of the list.

2- add_to_back(val):Adds a ewn node at the end of the list.

3- print_values():Traverses the list and prints all node values.
