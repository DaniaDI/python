###Zoo Management System
###Description:

This project is a simple object-oriented program that simulates a zoo system. It demonstrates key OOP concepts such as inheritance, polymorphism, and class relationships using animals like lions and tigers.
The system allows adding animals to a zoo, managing their attributes, and displaying their information.


###Objectives:
Practice Object-Oriented Programming (OOP) concepts
Implement inheritance and method overriding
Manage collections of objects inside a class
Build a simple zoo simulation system

###OOP Concepts Used
Inheritance: Lion and Tiger inherit from Animal
Polymorphism: Overriding the feed() method in child classes
Encapsulation: Each animal has its own attributes and behaviors
